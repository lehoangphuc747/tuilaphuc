Trong Python có các kiểu dữ liệu:
 - số nguyên (integers)
 - số thực (float)
 - chuỗi (string)
 - phân số (fraction)
 - số phức (complex)

Q: **1, 1.4, -123, 69.96,…** là kiểu dữ liệu nào?
